Software Architecture (UML)     Download (.jar, 15.2MB)


An Australian Stock Exchange monitor written in Java that retrieves and displays stock information in both real-time and time-lapsed. The focus of the software was to design and implement a robust, modular and extendable software architecture that could easily add or remove features. It connects to a web service provided by David Squire of Monash University. It was developed in partnership with a friend, Mark Sedrak.

An agent-based biological simulation of bumblebee's and their different harvesting techniques. With their environment as the varying parameter, this models aims to discover what combination of strategies yield the greatest results for the hive. This simulation and it's accompanying report was awarded Monash University's Top of Class for the Computer Science Project in Semester 1, 2012.

GPSTron is an adaption of the Tron Lightcycle game for Android tablets that uses GPS navigation for in-game movement. It uses XMPP for network communication and JSON for object serialisation. It connects to a private server provided by Robert Merkel of Monash University. It was developed in partnership with a friend, Mark Sedrak. The screenshot pictured above is from an incomplete version and we are expecting to open-source and properly document the project time permitting. There are no current plans to release it on the Android Play Store.
